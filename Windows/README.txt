This folder contains the software for Windows fo control Hubs.

Installation
============

- Install Python3
- Install bleak for Python3:
  pip install bleak
- Install pyside6 for Python3:
  pip install pyside6

Start the PC console
====================

- Open a CMD shell or a power shell
- Go into the directory Windows
  cd Windows
- Start the program
  python Pybricks_PcConsole.py
